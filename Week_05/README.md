# 学习总结week5

## AVL树、红黑树

> AVL：

平衡二叉搜索树：每个节点存balance factor= {-1,0,1}；通过四种旋转四种旋转操作维护树的平衡

不足：节点需要存储额外信息、且调整次数频繁

>  红黑树：

每个节点要么是红色、黑色

根节点黑色

每个叶子节点(空节点)是黑色

不能有相邻近的两个红色节点

从任意节点到其每个叶子的所有路径都包含相同数目的黑色节点

## 位运算

```
& 两位都为1，结果才为1
| 两位都为0，结果才为0
^ 两位相同为0，相异为1
~ 按位取反
<<  左移，高位丢弃，地位补0；*2
>>  右移高位补0：/2
x^0=x
两个数交换  c=a^b  a^c=b  b^c=a
将x最右边的n位清0  x&(-0<<n)
获取x第n位的值    (x>>n)&1
获取x的第n位的幂等值  x&(1<<n)
将第n位置为1     x|(1<<n)
将第n位置为0     x&(~(1<<n))
将x最高位至第n位(含)清零  x&((1<<n)-1)
```
